- **Generated by:** Tripo AI
- **Original size:** 218,988 verts / 437,988 faces (~29 MB)
- **Modified:** Decimated in Blender using the Decimate modifier to reduce
  file size for web delivery (now 10,943 verts / 21,898 faces, ~1.2 MB).
  **No topology problems were intentionally introduced or removed** — the
  decimation was strictly for bandwidth reasons.
- **Notes:** Still contains non-manifold edges, skinny triangles, degenerate
  faces, and an abnormal Euler characteristic. Used as a contrast sample in
  TopolGuard's Quick Start to show how AI-generated meshes may need manual
  inspection before use in production pipelines.
